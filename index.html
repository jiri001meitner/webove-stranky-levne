Kvalitní webové stránky přestavují výhodu pro Vaše podnikání. Pro jedince mohou být nepostradatelným propagačním a prezentačním nástrojem. Ačkoliv v dnešní době jsme obklopení webovými stránkami projektů, osob, korporací, neznamená to, že by byly samozřejmostí a že by byly zdarma. Naopakná, často se jedná o nákladnou záležitost a jejich vytvoření není pro člověka, který tomu příliš nerozumí, tak snadné.

Chcete vlastní web, tradičně byste měli

  1. mít vlastní doménu. Vlastní doména je absolutní základ, bez té jste OUT cena pár set Kč za rok
  2. mít zaplacený hosting, abyste mohli na něj nahrát web cena pár set Kč za měsíc
  3. nechat si vytvořit webové stránky cena x tisíc až desítek tisíc korun

Řešení je snadné

  • vyberte si design webu, jaký se vám líbí
  • pokud nemáte doménu, tak Vám pomohu s jejím výběrem a registrací
  • dodejte mi text pro webové stránky a logo ve formátu SVG, pokud nemáte, mohu něco vytvořit
  • vytvořím Vám web za rozumnou cenu, optimalizovaný pro mobilní zařízení a vyhledávače
  • zajistím Vám speciální plnohodnotný hosting, který nebudete platit užetříte x tisíc ročně
